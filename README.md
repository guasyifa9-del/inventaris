# Inventaris Kantor Kelompok 1
  
  Aplikasi manajemen inventaris kantor berbasis web
  
  ## Fitur
  - Manajemen barang
  - Peminjaman barang
  - Pengembalian barang
  - Laporan
  
  ## Teknologi
  - PHP
  - MySQL
  - Bootstrap
  
  ## Instalasi
  1. Clone repository
  2. Import database dari `inventaris_kantor.sql`
  3. Setting database di file `.env`
  4. Jalankan di localhost
